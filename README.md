🎓 KidKinder Website
📌 About the Project

KidKinder is a kids education website that includes multiple pages such as Home, About, Classes, and Contact. This project is built to showcase frontend development skills.

🚀 Features
Responsive Navigation Bar
Multiple Pages
Classes Section with Cards
Contact Form
Modal (Join Classes)
Footer with Social Links
🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5
Font Awesome

class.html

images/
